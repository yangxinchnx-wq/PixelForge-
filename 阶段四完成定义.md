# 阶段四完成定义

## 1. 文档目的

本文档定义"多图层 / 多区域 / 效果系统扩展"阶段的完成边界。

---

## 2. 阶段目标

将单图层渲染扩展为多图层混合渲染，引入区域边界和静态效果后处理管线，完善 Cache Key 三层失效机制。

---

## 3. 已完成项

### 3.1 Render IR V2 多图层扩展

已完成：
- **Layer.blendMode 字段** ✅
  - 支持 6 种混合模式（normal/multiply/screen/overlay/add/subtract）
  - StaticKeyInput 新增 `blendModes` 字段，blendMode 变更触发静态重编
- **Effect 接口与静态修饰符约束** ✅
  - Effect 含 id/type/params/targetLayer/targetRegion
  - FORBIDDEN_EFFECT_TYPE_NAMES 禁止动画类 type（animate/transition/motion/fade/pulse/flicker 等）
  - validateStaticBoundary() 递归扫描三重校验
- **Region 区域边界** ✅
  - Region.bounds（BoundingBox 归一化坐标）
  - region_eval.wgsl is_in_region() 区域裁剪
  - regionBuffer 传入 GPU

### 3.2 WGSL 多图层求值与混合

已完成：
- **region_eval.wgsl V2 重写** ✅
  - Uniforms 结构：resolution(vec2f) + seed(u32) + layerCount(u32)
  - descriptorBuffer 布局：长度 = 2 * layerCount（无 layerCount 前缀）
  - 多图层循环求值（从底到顶遍历）
  - blend_colors() 6 种混合模式实现
  - is_in_region() 区域边界检测
- **effect_post.wgsl 效果后处理** ✅
  - 5 种效果：blur/bloom/color_shift/vignette/mask
  - read_write storage texture 就地修改
  - 延迟创建管线（仅 IR 包含 effects 时触发）

### 3.3 工件编译 V2

已完成：
- **RegionCompileArtifact V2** ✅
  - schemaVersion = 'region-artifact-v2'
  - descriptorData：Uint32Array，长度 = 2 * visibleLayerCount
  - auxData：Float32Array，所有图层参数拼接
  - regionData：Float32Array，每 region 一个 vec4f
  - effectDescData：Uint32Array，effectCount + 效果描述符对
  - effectParamData：Float32Array，所有效果参数拼接
  - layers/regions/effects 结构化条目
  - visibleLayerCount / hasEffects 元数据
  - V1 兼容字段（layerId/opcode 指向第一个图层）
- **RegionCompiler V2** ✅
  - BLEND 作为图层 opcode 不再支持（抛 COMPILE_ERROR），通过 blendMode 实现
  - descriptorEntry 打包：opcode(8) | blendMode(8) | auxIndex(16)
  - 区域索引查找（0xFFFF = 无区域限制）
  - 效果参数编译（5 种效果类型）

### 3.4 三层 Cache Key 细化

已完成：
- **StaticKeyInput** ✅
  - canvas / opcodes / blendModes / effectTypes / outputStrategy / profileId
  - projectStaticKey() 投影函数含 blendModes
- **StructuralKeyInput** ✅
  - visibleFlags / layerOrder / regionBounds / regionOrder / layerRefs
- **DynamicKeyInput** ✅
  - paramEntries（layer + effect 参数，按 ownerId/paramKey 字典序排列）
  - seed / compileHints
- **MetadataKeyInput** ✅
  - layerSources / layerSourceRefs / paramOwnership
  - regionSources / regionSourceRefs / worldMetadata

### 3.5 文档对照审计

已完成：
- **骨架 §3.3 绑定组约定** ✅
  - 新增 binding 4（regionBuffer）
- **骨架 §4.5 迁移路径** ✅
  - descriptorBuffer V1→V2（长度 = 2 * layerCount，无 layerCount 前缀）
  - layerCount 通过 Uniforms 传入
- **骨架 §4.6.3 StaticKeyInput 定义** ✅
  - 新增 blendModes 字段
