## 5. 当前工作区历史态字段暴露结论

| 字段 | 展示位置 | 展示方式 |
|------|---------|---------|
| `frame` | TimelinePanel / FrameDataConsole | 帧编号 |
| `timestampMs` | FrameDataConsole | 时间戳 |
| `durationMs` | TimelinePanel / FrameDataConsole | 耗时 |
| `status` | CanvasWorkspace / InspectorPanel | 状态指示器 |
| `scenario` | CanvasWorkspace / InspectorPanel | 场景名 |
| `layerId` | CanvasWorkspace / InspectorPanel | 图层 ID |
| `opcode` | CanvasWorkspace / InspectorPanel | 指令名 |
| `patchId` | InspectorPanel / FrameDataConsole | 补丁编号 |
| `patchSummary` | InspectorPanel / FrameDataConsole | 补丁摘要 |
| `canvasSize` | CanvasWorkspace / InspectorPanel | 画布尺寸 |
| `outputFormat` | CanvasWorkspace / InspectorPanel | 输出格式 |
| `error` | CanvasWorkspace / InspectorPanel | 错误消息 |
| `artifactSchemaVersion` | FrameDataConsole | 工件版本 |
| `renderVerificationSnapshot` | FrameDataConsole | 一致性校验结果 |
| `compileHints` | FrameDataConsole → 原始载荷 | JSON 展示 |
| `capabilitySummary` | FrameDataConsole → 原始载荷 | JSON 展示 |

### 5.2 不需要额外暴露的字段

| 字段 | 原因 |
|------|------|
| `artifact` | 二进制 GPU 工件数据，非用户可读，仅用于回放内部 |
| `renderIrSnapshot` | IR 快照，非用户可读，仅用于回放内部 |
| `compileContextSnapshot` | 编译上下文快照，非用户可读，仅用于回放内部 |

### 5.3 结论

**当前工作区不需要暴露更多历史态字段。** 所有用户可读的帧记录字段均已有对应展示位置，内部数据（artifact / renderIrSnapshot / compileContextSnapshot）通过 FrameDataConsole 的"原始载荷"部分以 JSON 形式可查看。
