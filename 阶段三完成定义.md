# 阶段三完成定义

## 1. 文档目的

本文档定义"编辑器工作流与时间轴增强"阶段的完成边界。

---

## 2. 阶段目标

把当前验证型界面推进为更接近真实编辑器工作流的操作面。

---

## 3. 已完成项

### 3.1 时间轴增强

已完成：
- **时间轴缩放** ✅
  - 新增 `useTimelineViewport` composable，管理缩放级别（1x-10x）、帧宽度、内容宽度
  - 缩放按钮（+/−/1:1）和 Ctrl+滚轮缩放
  - 16 项 composable 单元测试
- **时间轴平移** ✅
  - 滚动容器 `overflow-x: auto`，浏览器原生滚动条
  - 滚轮水平滚动
  - 当前帧自动滚动到视口中央
- **区间选择** ✅
  - 标尺拖拽选择帧区间
  - Shift+点击扩展区间
  - 区间高亮覆盖层
  - "清除区间"按钮
  - 11 项区间选择单元测试
- **关键帧显示策略** ✅
  - 标尺上关键帧菱形标记
  - 关键帧剪辑按帧状态着色（ready=绿/error=红/idle=灰/initializing=黄）
  - 缩放级别 ≥1.5 时显示标签，否则显示色块
  - 时间轴头部显示关键帧计数
- **多轨道组织规则** ✅
  - 三条轨道：补丁轨道 → 关键帧轨道 → 验证轨道
  - 轨道标签显示帧数计数
  - 验证轨道按一致性状态着色（一致=绿/不一致=红）

### 3.2 工作流增强

已完成：
- **编辑帧与展示帧关系** ✅
  - 时间轴头部区分"编辑帧"和"展示帧"，展示帧用 accent 色高亮
  - 两者不同时同时显示，相同时只显示编辑帧
- **回放状态与编辑状态关系** ✅
  - 回放状态指示器（回放一致/回放异常）显示在时间轴头部
  - 成功=绿色徽章，失败=红色徽章
- **操作反馈** ✅
  - 标尺标记和剪辑添加 CSS transition（border-color/background/transform）
  - 按下时 scale(0.96) 反馈
- **工具栏分组** ✅
  - 播放组（上一帧/播放暂停/下一帧）
  - 缩放组（−/百分比/+/1:1）
  - 信息组（总帧数/清除区间）
  - 组间分隔线

### 3.3 数据面板增强

已完成：
- **摘要数据与技术数据分离** ✅
  - "摘要数据"：帧编号/时间戳/耗时/状态/补丁摘要
  - "技术数据"：补丁编号/工件版本/场景/图层/指令/画布/输出格式/错误信息
  - "一致性校验"和"原始载荷"保持独立
- **导出格式策略** ✅
  - 快照导出：仅摘要字段 JSON
  - 调试导出：完整帧数据 JSON
  - CSV 导出：表格格式
  - 完整导出：含面板数据 JSON（兼容旧格式）
  - 导出菜单 UI（下拉选择导出类型）
- **快照导出与调试导出明确区分** ✅
  - 快照导出 = 摘要字段（frame/timestampMs/durationMs/status/patchSummary/scenario/layerId/opcode）
  - 调试导出 = 完整 FrameSnapshot（含 payload/artifact/renderIrSnapshot 等）

## 6. 阶段三结论

**阶段三已经达到收口条件。**

- 时间轴从静态验证型轨道升级为可缩放、可平移、可区间选择的专业编辑器时间轴
- 工作流通过编辑帧/展示帧区分、回放状态指示器、工具栏分组和操作反馈变得自解释
- 数据面板通过摘要/技术分离和多种导出格式策略满足不同使用场景
- 192 项自动化测试全部通过

后续工作应进入阶段四：多图层/多区域/效果系统扩展。
