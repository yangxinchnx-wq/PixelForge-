# 图形运行验证项目

## 项目简介

这是一个基于 Vue 3、TypeScript、Vite、Tauri 和 WebGPU 的图形运行验证项目。

当前项目已经不再按“零散补功能”推进，而是按阶段管理。

目前已经完成：
- 阶段一：图形运行链路可行性验证

当前后续工作若继续推进，应进入：
- 阶段二：运行时稳定性与历史态一致性

---

- `项目阶段划分.md`

## 当前状态

当前已经具备：

- Render IR 与 Patch 协议基础设施
- PatchEngine 与 batch 回滚语义
- WebGPU 初始化与最小渲染链路
- 帧记录模型与统一适配层
- 时间轴与帧数据面板基础结构
- 历史帧回放入口 `replayFrame(frame)`
- 回放状态与版本校验
- 检查面板历史态投影
- 渲染签名一致性验证
- 完整的当前阶段测试基线

当前测试结果：
- `153` 项测试全部通过

## 开发命令

安装依赖：

```bash
npm install
```

启动开发环境：

```bash
npm run dev
```

构建项目：

```bash
npm run build
```

运行测试：

```bash
npm test
```
